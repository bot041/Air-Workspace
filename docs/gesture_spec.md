# AirScript Gesture Specification

| Gesture | Hand Shape | Action |
|---------|------------|--------|
| Draw | Index finger only | Draw strokes |
| Navigate | Index + Middle extended | Move without drawing |
| Select | Pinch (thumb + index) | Select top object under cursor |
| Drag | Pinch + move | Move selected object |
| Drop | Release pinch | Drop object; transfer if over tray |
| Partial Erase | Open palm swipe | Remove touched stroke points |
| Delete Object | Open palm hold 1 second | Delete entire object under cursor |

Cursor follows the index fingertip with the camera feed mirrored.
