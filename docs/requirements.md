# AirScript MVP Requirements

- Python 3.10+
- PyQt6
- OpenCV
- MediaPipe
- NumPy

## MVP Success Criteria

1. Open webcam feed.
2. Draw using index finger.
3. Reposition using two fingers.
4. Create multi-stroke objects.
5. Auto-finalize after 1 second of inactivity.
6. Select with pinch.
7. Drag naturally with offset.
8. Transfer to output tray.
9. Return objects to workspace.
10. Partially erase strokes.
11. Delete objects with palm hold.
12. Switch themes (Simple Ink / Neon AR).
13. Maintain 25–30 FPS.
