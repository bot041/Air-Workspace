AirScript MVP - PROJECT PLAN (Current)
=====================================

PROJECT OVERVIEW
----------------
AirScript is a desktop, fullscreen, gesture-controlled AR workspace for Windows.
It uses a webcam and computer vision to let users draw, move, and erase digital
ink in the air without touching a mouse, keyboard, or screen.

CORE VISION
-----------
- Live mirrored webcam feed visible at all times.
- Draw in the air using hand gestures.
- Preserve exactly what the user draws (stroke-level fidelity).
- No OCR or handwriting recognition.
- Single-hand interaction.
- Gesture-first workflow.
- Futuristic, glowing UI panels.

CURRENT FEATURES
----------------
1. Fullscreen workspace with mirrored webcam preview.
2. Real-time hand tracking via MediaPipe Hands.
3. Gesture-based drawing, navigation, selection, dragging, and erasing.
4. Six-color palette (green, red, neon, blue, orange, white) selectable by
   index-finger dwell or number keys 1-6.
5. Four brush modes: Ink, Neon, Marker, Highlighter.
6. Adjustable stroke thickness (+ / - keys).
7. Existing objects keep their color/brush/thickness when tools change.
8. Glowing, futuristic UI panels for colors and brushes.
9. Live gesture hint overlay.
10. FPS counter and status messages.
11. Smoke test and synthetic gesture unit tests.

TECH STACK
----------
- Python 3.10+

GESTURE SYSTEM
--------------
| Gesture | Hand Shape | Action |
|---------|------------|--------|
| Draw | Index finger only | Draw strokes |
| Navigate | Index + middle extended | Move without drawing |
| Select | Pinch (thumb + index) | Select top object under cursor |
| Drag | Pinch + move | Move selected object |
| Drop | Release pinch | Drop object |
| Partial Erase | Open palm swipe | Remove touched stroke segments |
| Delete Object | Open palm hold 1 sec | Delete entire object under cursor |

The cursor strictly follows the index fingertip and is mirrored so the workspace
feels like a mirror.

COLOR / BRUSH SYSTEM
--------------------
- Colors and brushes are selected from glowing side panels.
- Color/brush changes immediately apply only to new drawing.
- In-progress strokes are finalized first so they keep their original style.
- Brushes support opacity (Marker, Highlighter) and glow (Ink, Neon).

CONTROLS
--------
Gestures:
- Index finger      -> Draw
- Index + middle    -> Navigate
- Thumb + index     -> Select / Drag
- Open palm         -> Erase / hold to delete

Keyboard:
- 1-6               -> Select color
- F1-F4             -> Select brush (Ink/Neon/Marker/Highlighter)
- + / -             -> Increase / decrease thickness
- Esc               -> Quit

PERFORMANCE TARGETS
-------------------
- 25-30 FPS minimum.
- 640x480 hand tracking input for edge accuracy.
- Smooth real-time rendering.

ROADMAP
-------
- Save / load sessions and PNG export.

- Two-hand zoom / pan / rotate.
- Shape recognition.
- Sound / haptic feedback.
- Cross-platform packaging.
